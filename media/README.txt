# 媒体文件说明

## 文件夹用途
这个 `media` 文件夹用于存放您的自定义视频和音乐文件。

## 需要放入的文件

### 1. 视频文件
- **文件名**: `teletubbies.mp4`
- **说明**: 您的天线宝宝视频文件
- **格式**: MP4（推荐）
- **大小**: 建议50MB以内

### 2. 音乐文件  
- **文件名**: `happy-music.mp3`
- **说明**: 愉快的背景音乐
- **格式**: MP3（推荐）
- **大小**: 建议5MB以内

## 操作步骤
1. 将您的天线宝宝视频文件改名为 `teletubbies.mp4`
2. 将您的背景音乐文件改名为 `happy-music.mp3`
3. 将这两个文件放入此 `media` 文件夹
4. 刷新网页即可使用

## 文件命名要求
请确保文件名与上面列出的完全一致，否则需要修改 `script.js` 中的文件路径。
